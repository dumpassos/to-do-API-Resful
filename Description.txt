'To-Do'. API RestFul feita em NodeJs (com PostgreSQL) aonde podemos pegar, inserir, atualizar e remover tarefas (CRUD completo).

Foram utilizadas algumas bibliotecas, sendo as principais o Sequelize para conexão com banco de dados e o Express para criação de servidor.

npm install - para instalar as bibliotecas
npm start-dev - para iniciar o servidor
